# go-extensions

`go-extensions` is a collection of reusable extension packages for Go.

## extensions

- **caller** — [Documentation](./caller/README.md)
- **cast** — [Documentation](./cast/README.md)
- **event** — [Documentation](./event/README.md)
- **exchan** — [Documentation](./exchan/README.md)
- **tester** — [Documentation](./tester/README.md)

## Installation

Install only the package you need:

```bash
go get github.com/yourname/go-extensions/<package>
```

Example:

```bash
go get github.com/yourname/go-extensions/cast
```
